## 仿真作业1.1
输入电机基本参数
``` 
Va = 230;
Ia = 349;
Ra = 0.0144;
K = 1.225;
```

计算电机额定转矩
``` 
T0 = K*Ia;
```

计算转矩为420N·m时电机的转速
``` 
T1 = 420;
I1 = T1/K;
n1 = 60*(Va-I1*Ra)/(2*pi*K);
```
``` 
n2 = 1754.4;
```

计算转矩为420X(1+20%)N·m时电机的转速
``` 
T2 = 420*(1+0.2);
I2 = T2/K;
n2 = 60*(Va-I2*Ra)/(2*pi*K);
```
``` 
n2 = 1746.7;
```


计算转矩为420X(1-20%)N·m时电机的转速
``` 
T3 = 420*(1-0.2);
I3 = T3/K;
n3 = 60*(Va-I3*Ra)/(2*pi*K);
```


``` 
n2 = 1746.7;
```
